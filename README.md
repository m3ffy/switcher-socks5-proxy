# switcher-socks5-proxy

Простое браузерное расширение для быстрого переключения SOCKS5 прокси.

## Возможности
- Загрузка списка SOCKS5 прокси в расширение
- Переключение прокси одной кнопкой


## Как работает
1. Загрузите список прокси в окно расширения
2. Нажимайте кнопку для переключения на следующий прокси из списка


## Установка
1. Клонировать репозиторий
2. Открыть страницу расширений в браузере
3. Включить режим разработчика
4. Нажать «Загрузить распакованное расширение»
5. Выбрать папку проекта

Simple browser extension for fast SOCKS5 proxy switching.

## Features
- Load a list of SOCKS5 proxies into the extension
- Switch proxies with a single button click

## Как работает
1. Загрузите список прокси в окно расширения
2. Нажимайте кнопку для переключения на следующий прокси из списка

## Installation
1. Clone the repository
2. Open the browser extensions page
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
